# PHP-Maze
在地圖或迷宮裡面，走訪每一個地方，不會走回頭路，自行閃避不可通行的地方，直到抵達終點。

## Version 1.0 (2017-06-19 新增)
1. 透過 DFS (Depth-First-Search) 來走訪地圖。
2. 會自行閃避不可通行的地方，也不會超出地圖，直到抵達終點。
3. 目前沒有最佳解。
